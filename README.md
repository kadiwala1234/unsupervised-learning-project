# Unsupervised Learning Project

## Overview
This project applies unsupervised learning techniques such as clustering and dimensionality reduction to analyze a dataset.

## Dataset
- Name: Wine Dataset (UCI)
- Features: 13 numerical attributes
- Samples: 178
- No missing values

## Technologies Used
- Python
- Google Colab
- Libraries:
  - pandas
  - numpy
  - matplotlib
  - seaborn
  - scikit-learn

## Methods Used

### Clustering
- K-Means Clustering
- Hierarchical Clustering

### Dimensionality Reduction
- Principal Component Analysis (PCA)

## Results
- Optimal number of clusters: **3**
- PCA visualization shows clear cluster separation
- Silhouette score indicates good clustering performance

## How to Run the Project
1. Open the notebook in Google Colab
2. Run all cells step-by-step
3. View visualizations and results

## Key Learnings
- Importance of feature scaling
- How to choose optimal clusters
- Visualization using PCA
